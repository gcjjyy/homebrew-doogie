# Homebrew Tap for doogie-cli

두기의 고전게임 런처 CLI를 위한 Homebrew tap입니다.

## 설치

```bash
brew tap gcjjyy/doogie
brew install doogie-cli
```

## 사용법

```bash
doogie
```

## 요구사항

- DOSBox-X: `brew install dosbox-x`
